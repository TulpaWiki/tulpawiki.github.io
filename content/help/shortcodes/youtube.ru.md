---
title: youtube
description: Микрокод для получения для встраивания виджета видео YouTube
---
Данный микрокод возвращает виджет YouTube-видео на основе его идентификатора.

```markdown
{{</* youtube WGJzsKZ4oZ8 */>}})
```

Результатом работы микрокода станет следующее:

{{< youtube WGJzsKZ4oZ8 >}}
